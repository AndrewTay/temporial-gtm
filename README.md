# temporial-gtm

Go-to-market & beachhead test plan for **Temporial** — the non-human rigging wedge.

**Live:** https://andtay.com/temporial-gtm

## What this is
A single-page strategy doc covering:

- **The moat** — non-human topology (quadrupeds, birds, fish, insects, props) + good-enough motion, sold to speed/volume/emotion buyers. Why Mixamo, Meshy/Tripo, and Anything World leave it open.
- **The beachhead portfolio** — 3 tested in parallel:
  - **B1 · Non-human avatars** (VRChat / VTuber / furry) — *primary*, sell to commission artists.
  - **B2 · Indie game devs & jammers** — *parallel*, free Unity/Godot plugin + jam presence.
  - **B3 · Tabletop VTT tokens & minis** — *parallel*, batch idle-loop pipeline.
  - Held v2 options: animated pet digital twin (consumer breakout), AR filters, asset packs.
  - The objects/ecommerce **trap** and the only defensible slice (objects that move).
- **Measurement** — north-star *Non-human Rig Success Rate*, a 6-stage funnel, and the metrics that actually decide it.
- **An 8-week test plan**, pricing experiments, and a **scale / iterate / fold** decision gate.
- **Singapore advantages** (Japan timezone, cost edge, IMDA PIXEL / Gamescom Asia / NUS-NTU-SMU) and what to avoid.
